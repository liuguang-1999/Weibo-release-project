# 微博发布案例

## 渲染步骤解析

### 业务逻辑

1. 第一步向(谷歌浏览器)、的服务器 获取数据 
2. 获取到数据后进行JSON 转换
3. 转换后进行渲染 添加到页面上
4. 最后进行代码封装 方便接下来的调用和使用

```js
// 加载函数渲染页面 
laoding();
    function laoding() {
        // 向服务器 获取数据
        var laod = localStorage.getItem("jinqu")
        if (laod !== null) {
            var arr = JSON.parse(laod) // JSON 转换
             // 数据遍历 渲染到节点上
            arr.forEach((itme) => {
                 // 创建节点 添加数据
                str = `<li><p>${itme.neirong}</p><span data-sf=${itme.sf}>删除</span></li>`
                 // 把创建好的数据 添加DOM到节点里
                $("#list").append(str)
            })
        }
    }
```

## 向服务器发送存储数据

### 业务逻辑

1. 点击发送按钮 
2. 获取文本框值 创建标签 把文本框值放入标签中  最后把 标签添加到节点中
3. 清空页面中文本框里的值
4. 重新渲染页面

```js
// 向服务器发起存储数据
	var list = [];
    $("#btn").click(function () {
        // ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 本地里操做
        // 创建一个 唯一的数字信息 方便做删除 
        var sf = Math.random() + "_" + Date.now()
        var va = $("#tet").val()
        var str = `<li><p>${va}</p><span data-sf=${sf}>删除</span></li>`
        $("#list").prepend(str)
        $("#tet").val("")
        // ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 服务器存储操做
        var obj = {
            neirong: va,
            sf: sf
        }
        // 把obj这单独的一条信息添加到list数组里面去
        list.unshift(obj)
        // 把list 转换成 JSON 字符串
        var arr = JSON.stringify(list)
        // 向服务器发送带有JSON字符串的 存储数据
        localStorage.setItem("jinqu", arr)
        // 重新渲染
        laoding()
    })
```

## 向服务器发送删除数据

### 业务逻辑

1. 建议用事件委托做点击事件 因为删除按钮是动态生成的
2. 点击删除按钮后获取 删除按钮 里面的自定义属性
3. 循环找出list 数组里面对应的唯一 数字信息 进行删除
4. 删除后重新添加到 (谷歌服务器里面)

```js
// 删除操做
	// 父元素做事件委托 冒泡到子元素span上
    $("#list").on("click", "span", function () {
        // 获取 唯一数字信息
        var val = $(this).attr("data-sf")
        // 遍历查找 list数组里面和val 相同的 唯一数字信息
        for (var i = 0; i < list.length; i++) {
            if (list[i].sf == val) {
                // 找到后进行删除
                list.splice(i, 1);
                // 找到后进行删除 循环就不需要在往下进行了
                break;
            }
        }
        // 发起存储
        var arr = JSON.stringify(list)
        localStorage.setItem("jinqu", arr)
        // 重新渲染
        laoding();
    })
```

## 按回车进行发送信息

### 业务逻辑

1. 给文本域 做键盘事件
2. 事件里面写入 前的点击事件

```js
 // 点击回车 发送数据
    tet.onkeydown = (e) => {
        // 判断 是否 按的是 回车键
        if (e.keyCode == 13) {
            // 调用 之前的点击事件
            $("#btn").click();
        }
    }
```
